Código-fonte

O comando document.getElementById('teste') é escrito em JavaScript, e esse efeito de fonte é feito usando a tag <code>.

Para colocar um código indentado, basta usar a tag <pre>.

            
int main() {

    int number;
    scanf("%d", &number);

    if(number % 2 == 0)
        printf("O número é par.\n");

    else
        printf("O número é ímpar.\n");
    
    return 0;
}
