Principais Formatações

Negrito - Destaque

Nesta frase, temos um termo em destaque usando a tag <b>, não semântica.

Já nesta frase, temos um termo em destaque usando a tag <strong>, semântica.

Itálico - Ênfase

Nesta frase, temos um termo em ênfase usando a tag <i>, não semântica.

Já nesta frase, temos um termo em ênfase usando a tag <em>, semântica.

Texto marcado

Podemos criar, também, um texto marcado usando a tag <mark>.

E aqui, temos outro texto marcado usando a tag <mark>.

Texto grande e pequeno

Estamos criando um texto grande, usando a tag <big>, e um texto pequeno, usando a tag <small>, neste parágrafo.

Texto deletado

Podemos marcar um texto como excluído, com a tag <del>, para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido, com a tag <ins>, para dar uma ênfase e indicar que ele foi adicionado depois.

Outra forma de marcar um texto sublinhado é usando uma tag não semântica, a tag <u>.

Texto sobrescrito

Para inserir coisas do tipo x20 + 3, basta utilizar a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O basta utilizar a tag <sub>
