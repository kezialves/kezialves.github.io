Lista de Definições

Para criar uma lista de definições, basta usar a tag <dt> para o termo e a tag <dd> para sua descrição.

Kon do Aki
Aki Hayakawa tinha um contrato com o Fox Devil. Em troca de alimentá-lo com uma parte de seu corpo, Aki poderia usar seu poder em combate levantando a mão e dizendo a palavra “Kon”, fazendo com que a cabeça do Fox Devil parecesse morder instantaneamente o corpo do alvo.
Kon do Megumi
A Energia Amaldiçoada de Megumi concede a ele a possibilidade de usar a técnica das Dez Sombras. Ela permite que Megumi crie Shikigamis a partir das sombras, entre eles, os Gyokuken (Cães Divinos) e o Gyokuken - Kon (Cão Divino: Totalidade), que é uma versão do Cão Divino preto após ter absorvido a energia amaldiçoada do cão branco.
